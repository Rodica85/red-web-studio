# RED — Brand Guidelines

## Logo Files
- `logo-primary.svg` — Main logo (light background)
- `logo-dark.svg` — Logo on dark background
- `logo-icon.svg` — Icon/favicon (circle with R)

## Brand Colours

| Colour | Hex | Usage |
|--------|-----|-------|
| **RED Primary** | #E63946 | Main brand colour, CTAs, accents |
| **RED Dark** | #C1121F | Hover states, gradients |
| **Dark Navy** | #1A1A2E | Backgrounds, text |
| **Charcoal** | #2D2D44 | Secondary backgrounds |
| **Light Grey** | #F8F9FA | Page backgrounds |
| **White** | #FFFFFF | Cards, content areas |
| **Text Grey** | #555555 | Body text |

## Typography

| Usage | Font | Weight |
|-------|------|--------|
| **Headlines** | Inter / Arial Black | 800-900 |
| **Subheadings** | Inter / Helvetica Neue | 600 |
| **Body** | Inter / Helvetica Neue | 400 |
| **Buttons** | Inter / Helvetica Neue | 600 |

## Tone of Voice
- **Professional** but approachable
- **Clear** — no jargon
- **Confident** — we know what we're doing
- **Helpful** — focused on the client's success
- **British English** — colour, optimise, programme

## Tagline Options
1. "We Build. You Grow." (primary)
2. "Websites That Work For You"
3. "Your Business, Online. Done Right."
